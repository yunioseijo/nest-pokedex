# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar
```
$ yarn install
```
3. Tener Nest CLI instalado
```
$ npm i -g @nestjs/cli
```
4.Levantar la base de datos
```
$ docker-compose up -d
```
## Stack usado
* MongoDB
* Nest

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
